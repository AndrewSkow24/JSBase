Введение в язык JavaScript для новичков

Язык JavaScript предназначен для выполнения в браузере наряду с HTML и СSS. Но если эти языки предназначены для вёрстки структуры сайта, то JavaScript позволяет оживлять web-страницы - делать их реагирующими на действия пользователя или демонстрировать некоторую динамичность (к примеру, смена картинок в блоке или красиво плавно выпадающие меню).
