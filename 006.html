Подключение файла со скриптами JavaScript

JavaScript-код можно писать в отдельном файле, который затем будет подключён к HTML-файлу. Давайте посмотрим как это делается.

Для начала создадим нам файл со скриптом. У этого файла должно быть расширение .js. Для примера назовём его "006.js". Разместим в нём какой-нибудь код: alert("....");
